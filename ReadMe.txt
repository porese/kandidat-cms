УСТАНОВКА
-------------------------------------------------
1. Закачать на сервер содержимое папки www (для соответствующей редакии) в бинарном режиме.
2. Набрать в браузере адрес http://ВАШ_САЙТ/admin/index.php
При этом должно открыться окно с формой авторизации, указать Логин: admin и Пароль: admin и зайти в панель управления сайтом. Перейти в раздел настройки и задать свои данные. 
3. После смены пароля зайдите с новым паролем, и добавьте свои страницы.
Страницы сохраняются в папку articles , поэтому у неё тоже должны быть выставлены права на запись, а также права на запись должны быть и у файла engine/menu.php
5. Задайте свой емал адрес в файле mycode/feedback.php (строка $mail_target = 'имя@ваш_домен.ru';)
6. Если на каком то хостинге скрипт не хочет работать, то открыть в редакторе файл engine/functions.php и прочитать вверху подсказку.
7. РЕКОМЕНДУЕМ закрыть паролем папку admin средствами защиты на сервера хостинга.

Оформление и шаблоны
-------------------------------------------------
Бесплатных шаблонов очень много. По аналогии создать их очень просто. Дополнительны темы оформления Вы сможете найти на нашем сайте
http://kan-studio.ru/

Вставки-переменные в шаблонах
<?php echo $sitename;?>  - имя сайта
<?php echo $sitetitle;?>     - заголовок страницы
<?php echo $metadescription;?> - мета-тег description
<?php echo $metakeywords;?> - мета-тег keywords
<?echo $contentcenter;?>  - центральное содержание
<? include $localpath .'/engine/menu.php'; ?> - меню ссылок на страницы
<? include $localpath .'/mycode/lastnews.php'; ?> - вывод блока краткого содержания последней новости
<? include $localpath ."/mycode/$myinclude.php"; ?> - ваш php-код на страницы
<?php echo $lnav;?> - визуальное отображение глубины навигации по сайту
-------------------------------------------------
СОВЕТЫ:
-------------------------------------------------
1. Папка mycode , для включения вашего php-кода для любой страницы. По умолчанию на всех страницах включен php-код из файла mycode/main.php (просто вывод текущей даты). Имя кода для каждой страницы можно менять из админки (не забудьте сперва создать свои коды и положить в эту папку) или вручную, используя редактор, прописав в файле страницы (к примеру страница articles/page1.dat) вместо
<!-- Kan_myinclude -->main<!-- Kan_myinclude --> 
другое имя включаемого файла
<!-- Kan_myinclude -->page<!-- Kan_myinclude -->
и тогда для этой страницы код будет включен из файла mycode/page.php
СОВЕТ: в будущем этой папке лучше задать уникальное имя в целях безопасности.

2. При добавлении картинок (через редактор nicEdit) в содержимое страницы права на запись папке media выставлять не требуется (скрипт всё сделает сам).

3. Если необходимо вставить код для SAPE, читайте текстовой файл в этом архиве kod-sape.txt

Вообще то не хотел говорить, но скажу так: Вы когда скрипт примериваете на локальном компьюторе на предмет функционала, то, собстенно, это и есть ваша копия будущего сайта, ... а значит, по-мне, дак админку иметь на сервере хостинга нет смысла - все файлы страниц (новые или измененые на локалке) из папки articles и конфиг с менюшкой легче заливать напрямую через ftp-клиента с заранее собранного, так сказать, варианта. Плюс в этом случае ещё и в том, что если нет админки на хостиге, значит и взламывать нечего желающим пошекотать вам нервы. А про саттелиты - так и так - залил и забыл.
